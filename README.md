# hw8.py
В этом задании была создана мини база данных с использованием SQLite. Реализованы две связанные таблицы (customers и orders) с отношением один ко многим. Добавлены функции для вставки данных, выполнения JOIN-запросов, агрегатных функций (COUNT, MAX, AVG), GROUP BY, подзапросов и VIEW для удобного просмотра связанных данных.
